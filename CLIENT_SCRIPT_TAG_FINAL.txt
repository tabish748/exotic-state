IMPORTANT NOTES:

1. CORS Configuration:
   Make sure the client's domain is added to ALLOWED_ORIGINS
   on the server. Update the .env file on EC2:
   
   ALLOWED_ORIGINS=https://www.exoticestates.com,https://exoticestates.com

2. HTTPS (Recommended for Production):
   For production, you should set up HTTPS/SSL. Options:
   - Use Nginx reverse proxy with Let's Encrypt SSL
   - Use AWS Application Load Balancer with SSL certificate
   - Or use CloudFlare for SSL termination

3. Domain Name (Optional but Recommended):
   Instead of IP address, set up a domain name:
   - Point domain to EC2 IP: 16.16.128.91
   - Update script tag to use domain name
   - Example: chatbot.exoticestates.com
