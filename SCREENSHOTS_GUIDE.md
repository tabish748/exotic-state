## üì∏ Screenshot Specifications

### Technical Requirements

- **Format:** PNG (preferred) or JPG
- **Resolution:** Minimum 1920x1080 for desktop, actual device resolution for mobile
- **Quality:** High quality, clear text
- **File Size:** Optimized but readable
- **Naming:** Use exact filenames specified above

### Best Practices

1. **Clean Screenshots:**
   - Remove personal information
   - Hide sensitive data
   - Use test/demo data only

2. **Consistent Styling:**
   - Same browser for consistency
   - Same zoom level (100%)
   - Clean browser (no extensions visible)

3. **Good Composition:**
   - Center important elements
   - Show full context
   - Include relevant UI elements

4. **Annotations (Optional):**
   - Add arrows/boxes to highlight features
   - Add text labels if helpful
   - Keep annotations minimal and professional
