INTEGRATION EXAMPLES:
---------------------

HTML PAGE:
----------
Add before closing </body> tag:

<script src="https://YOUR-AWS-URL/public/chatbot-widget.js"
        data-api-url="https://YOUR-AWS-URL/api"
        data-config='{"theme": "light", "position": "bottom-right"}'></script>

VUE.JS:
-------
In your Vue component or layout:

mounted() {
  const script = document.createElement('script');
  script.src = 'https://YOUR-AWS-URL/public/chatbot-widget.js';
  script.setAttribute('data-api-url', 'https://YOUR-AWS-URL/api');
  script.setAttribute('data-config', JSON.stringify({
    theme: 'light',
    position: 'bottom-right'
  }));
  document.body.appendChild(script);
}

NEXT.JS:
--------
In _app.js or layout component:

useEffect(() => {

  script.src = 'https://YOUR-AWS-URL/public/chatbot-widget.js';
  script.setAttribute('data-api-url', 'https://YOUR-AWS-URL/api');
  script.setAttribute('data-config', JSON.stringify({
    theme: 'light',
    position: 'bottom-right'
  }));
  document.body.appendChild(script);
}, []);

TESTING:
--------
After adding the script tag:

1. Open the page in your browser
2. Look for the chat button in the bottom-right corner
3. Click the button to open the chat window
4. Send a test message
5. You should receive an AI response

If the chatbot doesn't appear:
- Check browser console (F12) for errors
- Verify the script URL is correct
- Contact support if issues persist
